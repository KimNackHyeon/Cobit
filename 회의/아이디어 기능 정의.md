# 아이디어 기능 정의

### **웹 기반 코딩 교육 플랫폼**

- 로그인 / 회원가입
- 단계
- 코딩 모험 - 스토리
    - 미션
        - 미션 해결 방법
            1. STT(Speech To Text)
                - 생각한 것을 실행하기
                    - 성공시 : 다음 단계
                    - 실패시 : 다시 말하기
            2. 코딩 블록
                - 코딩 블록 작성 후 실행하기
                    - 성공시 : 코드화 / 별 1~3개
                    - 실패시 : 다시하기
    - 자유 필드
        - 모든 기능 다 가능
    - 모션 인식
        - 자신의 얼굴로 된 캐릭터 > 캐릭터화
        - 나만의 캐릭터 만들기
        - 나의 행동을 모사하는 캐릭터
        - 캐릭터와 함께 사진찍기
